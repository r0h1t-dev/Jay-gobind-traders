### Proprietor Photos
To add photos for Rakesh Kumar Chaudhary and Yugal Kishor Chaudhary:

1. Create an `images` folder in this directory
2. Add photos named `rakesh.jpg` and `yugal.jpg`
3. In `index.html`, find the `.proprietor-photo` divs and replace them with:

```html
<div class="proprietor-photo">
    <img src="images/rakesh.jpg" alt="Rakesh Kumar Chaudhary" style="width: 100%; height: 100%; object-fit: cover; border-radius: 50%;">
</div>
```

### Business/Gallery Photos
To add business photos:

1. Add your photos to the `images` folder
2. Find the `.gallery-item` divs in `index.html`
3. Replace the placeholder content with:

```html
<div class="gallery-item">
    <img src="images/your-photo.jpg" alt="Description" style="width: 100%; height: 100%; object-fit: cover;">
</div>
```

### About Section Photo
For the main business photo in the About section, replace the `.about-image-placeholder` div with an actual image.
